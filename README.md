## What is gor?
gor which stands for "Green or Red", shows your test result with color as a filter.
Helps your TDD life.

## Requirements
- [Mosh 0.2.4](http://code.google.com/p/mosh-scheme/downloads/list) or higher

## Using gor
    % run-your-test | gor
